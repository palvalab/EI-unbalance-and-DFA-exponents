MATLAB CODE

System requirements:
This code requires Matlab software. The code has been tested on versions R2017, R2018 and R2020 on Windows platforms (Windows 10 Enterprise and Windows Server 2019 Standard). 
Installation of Matlab should take less than 1 hour.

Instructions:
The instructions below produce the main results of manuscript
Download the dataset from [https://datadryad.org/stash/dataset/doi:10.5061/dryad.wwpzgmspw] and save the files 'DFA_fEI_estimates.mat' and 'Figure 1.zip' to a folder on your computer. In the matlab script (Figure_1), set varaiable 'rsdata' to this folder and also unzip the file.
The scripts replicate the main findings of the study which are shown in Figures 1b-e.
Repaeat the procedure, but now for Figure 2c-f.
Codes for other figures can be found at https://github.com/palvalab/EI-unbalance-and-DFA-exponents.
